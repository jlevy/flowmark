# Code Blocks

A backtick fenced code block:

```python
def hello():
    print("Hello, world!")
    x = 'single quotes'
    y = "double quotes"
    z = "it's a test..."
```

A tilde fenced code block:

~~~javascript
function greet() {
    console.log("Hello...");
    const msg = 'Don\'t worry';
}
~~~

Text after code blocks with "quotes" and ... ellipses.
